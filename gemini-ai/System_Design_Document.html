üîÑ Data Flow & API Design

Request Flow

1. User Input: User types message in React frontend

2. API Call: Frontend sends POST request to /api/generate

3. Processing: Express server validates and processes request

4. AI Request: Server calls OpenAI API with formatted prompt

5. AI Response: OpenAI returns generated text

6. Response: Server sends formatted response back to frontend

7. Display: Frontend renders AI response with markdown

API Endpoints

Method Endpoint Description Request Body Response
POST /api/generate Generate AI response { taskType, prompt } { text }

Request/Response Schema

// Request Body { "taskType": "general" | "daily" | "code", "prompt": "User's input message" } // Success Response { "text": "AI generated response in markdown format" } // Error Response { "error": "Error message description" }
